# Web计算器CI/CD项目

这是一个用于学习CI/CD和蓝绿部署的Web计算器项目。

## 功能特性

- 基础计算功能：加、减、乘、除
- RESTful API设计
- 完整的CI/CD流水线
- 蓝绿部署策略
- 健康检查
- 自动化测试
